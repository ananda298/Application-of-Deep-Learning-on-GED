To run the code, download the zip file and unzip all of it. 
The code is made to run on Google Collab. 
Import the jupyter notebook file, as long as the train and test dataset folders. 
Take note of the train and test file location, to copy and paste into the code and replace
the location in the code. 
Use run all, to get the results. 
All the different convolution and activation layers are highlighted using #. 
To test the different results, unhighlight accordingly.
